# Snack Game (Python + pygame)

This is a small Snake-style game implemented in Python using pygame.

Quick start (Windows PowerShell):

```powershell
python -m pip install -r requirements.txt
python snack_game.py
```

Controls:
- Arrow keys: move
- R: restart after Game Over
- Esc: quit

If you see an error about installing pygame, try installing a binary wheel for your Python version or visit https://www.pygame.org/wiki/GettingStarted
